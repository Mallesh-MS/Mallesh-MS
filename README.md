## Featured Projects

### Real-Time Athlete Monitoring Dashboard
> A dual-tier IoT ecosystem to stream and visualize live athletic biometric and movement data.

- **Edge Hardware** — ESP32 processing MPU6050 (motion, squat count, fall detection) and MAX30102 (Heart Rate & SpO₂)
- **Architecture** — Local WebSocket connection (HTML5/CSS3/Chart.js) combined with a production-grade Node.js gateway and React cloud client
- **Key Features** — Auto-reconnect with exponential backoff, real-time rolling statistics, CSV/JSON telemetry exporter

> A full-stack medical analysis platform for MRI/CT scan parsing and brain tumor classification.

- **Frontend** — SPA using `cornerstone-core` and `dicom-parser` to render 16-bit diagnostic `.dcm` files in-browser
- **Backend** — Flask API gateway supporting both JSON base64 and `multipart/form-data` uploads
- **AI Integration** — Google Gemini AI acting as a patient-friendly diagnostic explainer
